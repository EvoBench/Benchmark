    DTLZ1

    Definição

    O problema DTLZ1 possui uma frente ótima de Pareto formada por um hyperplano linear:

    \[ \textcolor{blue}{\boldsymbol{f_1(x)+f_2(x)...+f_{M-1}(x)+f_M(x) = 0.5}}\text{ definido por} \Large \sum_{m=1}^M = \textcolor{blue}{\boldsymbol{f_{m}}}^* \]

    DTLZ1 é definida pelas seguintes variáveis:

    • M: Número de objetivos do problema
    • N: Número de variáveis de decisão
    • K: representado por:
    \[ \boldsymbol{K}\text{ é o valor do vetor } \boldsymbol{\big|X_M\big|} \text{ onde todos } \boldsymbol{X_i} \text{ pertencem a } \boldsymbol{X_M}\text{ }\boldsymbol{(X_i \in X_M)} \]

    O valor de N pode ser obtido pela equação N = K+M-1 substituindo o valor de K. Alternativamente, definindo o valor de K pela equação K = N-M+1, e substituir o valor de N.
